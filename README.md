# SynapseLogic - Análisis de Conectividad Neuronal

Herramienta para modelar redes neuronales como grafos dirigidos, calcular rutas de activación (Dijkstra), detectar zonas aisladas (BFS/DFS) y simular fatiga sináptica. Implementa tabla hash propia y visualización con GraphStream.
